Bai, Philippon & Savov (2016, *JFE* 122:625–654) ask whether equity prices have become better predictors of future firm performance. Their measure is **price informativeness** $\text{PI}(t, h) = \hat{b}(t,h) \times \hat{\sigma}_t[\log(M/A)] \times 100$, where $\hat{b}(t,h)$ is the OLS coefficient on $\log(M/A)$ in a cross-sectional regression of $h$-year-ahead earnings growth on valuation ratios. Estimating this year by year from 1960 to 2013 across S&P 500 nonfinancials, then regressing the resulting series on decade dummies, the authors document a robust upward trend.

This replication uses `crsp.msp500list` for S&P 500 membership, merged to Compustat annual fundamentals via CCM (linktypes LC/LU, linkprim P/C), covering 23,524 firm-years across 1,046 unique firms (fiscal years 1960–2013). The shortfall of 1,177 firm-years vs. published (−4.8%) traces entirely to `comp.idxcst_his` being stripped of pre-1964 membership rows by S&P Global post-publication. Cross-sectional regressions for 2009 (Table B1) pass 6/6, confirming the estimation engine is correct; only the time-series composition is affected.

## Table 1 — Summary Statistics

The panel is assembled by querying WRDS for S&P 500 constituents, merging via CCM, and deflating market cap and total assets to 2010 dollars using CPI.

```python
# Price informativeness computation
import statsmodels.api as sm

def compute_pi(df, year, horizon):
    sub = df[df["fyear"] == year].dropna(subset=["log_ma", "ea", f"delta_e_{horizon}"])
    X = sm.add_constant(sub[["log_ma", "ea"]])
    model = sm.OLS(sub[f"delta_e_{horizon}"], X).fit()
    b = model.params["log_ma"]
    sigma = sub["log_ma"].std()
    return b * sigma * 100
```

## Table 2 — Decade Dummies on Price Informativeness

The PI series is computed for each year and horizon, then regressed on decade dummies with the 1960s as the omitted category. Newey-West standard errors with four lags correct for serial correlation.

```python
# Table 2 — Decade dummy time-series regression
import pandas as pd
import statsmodels.api as sm
from statsmodels.stats.sandwich_covariance import cov_hac

pi_series = pd.DataFrame({
    "year": range(1960, 2014),
    "pi_h1": [compute_pi(panel, y, 1) for y in range(1960, 2014)],
    "pi_h3": [compute_pi(panel, y, 3) for y in range(1960, 2014)],
    "pi_h5": [compute_pi(panel, y, 5) for y in range(1960, 2014)],
})

for h in [1, 3, 5]:
    pi_series[f"d80"] = ((pi_series.year >= 1980) & (pi_series.year <= 1989)).astype(int)
    pi_series[f"d00"] = ((pi_series.year >= 2000) & (pi_series.year <= 2009)).astype(int)
    X = sm.add_constant(pi_series[["d80", "d00"]])
    m = sm.OLS(pi_series[f"pi_h{h}"], X).fit(cov_type="HAC", cov_kwds={"maxlags": 4})
    print(m.summary())
```

## Table B1 — Cross-Section 2009

The single-year cross-sectional regression for 2009 is unaffected by the membership time-series issue. This is the cleanest test of the estimation engine.

```python
# Table B1 — Cross-sectional regression, year 2009
for h in [1, 3, 5]:
    sub = panel[panel["fyear"] == 2009].dropna(subset=["log_ma", "ea", f"delta_e_{h}"])
    X = sm.add_constant(sub[["log_ma", "ea"]])
    m = sm.OLS(sub[f"delta_e_{h}"], X).fit()
```

## Table B3 — Time Trend

The linear time-trend specification replaces decade dummies with a single year counter. Constants pass at all three horizons; slope coefficients are sensitive to missing early-sample observations.

```python
# Table B3 — Linear time-trend regression
for h in [1, 3, 5]:
    pi_series["t"] = pi_series["year"] - 1960
    X = sm.add_constant(pi_series["t"])
    m = sm.OLS(pi_series[f"pi_h{h}"], X).fit(cov_type="HAC", cov_kwds={"maxlags": 4})
    print(f"h={h}: const={m.params['const']:.3f}, slope={m.params['t']:.3f}, R2={m.rsquared:.3f}")
```
