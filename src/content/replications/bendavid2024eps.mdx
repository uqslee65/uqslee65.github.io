Ben-David & Chinco (2024) test whether corporate managers optimize earnings per share rather than firm value. The key identification exploits a mechanical threshold: when a firm's earnings yield $\text{EY} = \text{EPS}/P$ exceeds the Moody's Aaa corporate bond rate, adding leverage arithmetically raises EPS, giving an EPS-maximizing manager a discrete incentive to borrow. The excess earnings yield is $\text{ExcessEY}_{n,t} \equiv \text{EY}_{n,t} - r^{\text{Aaa}}_t$, and the binary indicator $\mathbf{1}[\text{IsValue}_{n,t}]$ switches on when this quantity is positive.

This replication uses a 2026 WRDS pull (52,274 firm-years across 6,012 firms, 1976–2023) merged from IBES consensus forecasts, Compustat fundamentals, and CRSP monthly returns via CCM linkage. All 11 gate checks pass; coefficient magnitudes run approximately 17% above published values due to WRDS retroactive sample backfill between the paper's ~2022 data pull and the 2026 extraction.

## Table 1 — Summary Statistics

Sample construction pulls from WRDS; the `build_eps_panel` function merges IBES, Compustat, and CRSP and computes the threshold indicator.

```python
# Data construction
import wrds
db = wrds.Connection()

# Merge IBES consensus, Compustat, CRSP via CCM
panel = build_eps_panel(db, start=1976, end=2023)
panel["excess_ey"] = panel["ey"] - panel["aaa_rate"]
panel["is_value"] = (panel["excess_ey"] > 0).astype(int)
```

## Table 2 — Financial Leverage

The dependent variable is the change in book leverage ($\Delta \text{Debt/Assets}$, in percentage points) from year $t$ to $t+1$. Columns add progressively richer fixed effects and firm-level controls. Standard errors are double-clustered by firm and fiscal year.

```python
# Table 2 — Financial leverage response
import pyfixest as pf

specs = {
    "(1) No FE": "delta_leverage ~ is_value",
    "(2) Year FE": "delta_leverage ~ is_value | fyear",
    "(4) Year + Controls": "delta_leverage ~ is_value + log_at + bk_lev + profitability + mtb + tangibility + sales_growth | fyear",
}
for name, fml in specs.items():
    m = pf.feols(fml, data=panel, vcov={"CRV1": "gvkey+fyear"})
    print(f"{name}: β={m.coef()['is_value']:.2f}, t={m.tstat()['is_value']:.1f}")
```
